# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-07

### Added

- Initial release of the terraform-aws-ec2-instance module
- EC2 instance provisioning via launch templates
- IAM instance profile creation with SSM and CloudWatch policies
- IMDSv2 enforcement by default
- EBS-optimized instances with encrypted root volumes
- Configurable additional EBS volumes with independent settings
- Security group creation with customizable ingress rules
- Support for user data and base64-encoded user data
- Spot instance sub-module with configurable spot options
- Basic, advanced, and complete usage examples
- Comprehensive variable validation
- Full documentation and README
